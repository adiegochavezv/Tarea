# Proyecto: Formulario Validado

## ¿Cómo abrir el proyecto?
1. Abre la carpeta principal del proyecto.
2. Haz doble clic en el archivo `index.html`.
3. El formulario se abrirá en tu navegador web (Chrome, Edge, etc.).

## Checklist
- [x] Nombres de carpetas correctos (`assets/css`, `assets/js`, `assets/img/capturas`).
- [x] Formulario con nombre, email, select y checkbox.
- [x] Etiquetas `label` asociadas a cada input.
- [x] Bloquea el envío si hay campos inválidos.
- [x] Error si el email está vacío.
- [x] Error si el email no tiene `@`.
- [x] Error si el checkbox no está marcado.
- [x] Muestra mensaje de confirmación si todo está OK.
- [x] Capturas de prueba guardadas en la carpeta correspondiente.
